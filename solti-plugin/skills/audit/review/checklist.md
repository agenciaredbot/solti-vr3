# Audit Checklist

Use this checklist during Pass 2 (AI analysis) to ensure nothing is missed.

## CRM
- [ ] Contact count trending up week-over-week?
- [ ] New leads being qualified within 48h?
- [ ] Hot leads (score >= 80) getting immediate attention?
- [ ] Any contacts stuck in same status > 14 days?
- [ ] Email/phone coverage > 50%?
- [ ] Score distribution reasonable (not all 0 or all 100)?

## Campaigns
- [ ] Active campaigns have recent events (within 24h)?
- [ ] Bounce rate < 5% on all campaigns?
- [ ] Open rate > 20% on email campaigns?
- [ ] Reply rate tracked and improving?
- [ ] No campaigns stuck in SENDING > 48h?
- [ ] Failed campaigns investigated?

## Services
- [ ] All credentials valid (tested within last 7 days)?
- [ ] No repeated API errors in recent logs?
- [ ] WhatsApp instances connected?
- [ ] Brevo sender domain verified?

## Costs
- [ ] Daily spend within budget?
- [ ] Cost per lead < target ($X)?
- [ ] No unused services consuming credits?
- [ ] Credit balance sufficient for next 7 days?

## Security
- [ ] API keys rotated within last 90 days?
- [ ] No credentials exposed in logs?
- [ ] Plugin API key working correctly?
